React + TypeScript + Vite + Tailwind CSS + Zustand

1. Канбан доска /kanban

Четыре колонки: To Do / In Progress / Review / Done
Карточки задач в колонках: название, ответственный (аватар + имя), дедлайн, приоритет (low / medium / high)
На карточке кнопки "←" и "→" для перемещения между колонками
Счётчик карточек в каждой колонке
Фильтр по ответственному
Клик на карточку → модалка с деталями задачи (название, описание, подзадачи, комменты)


2. Центр уведомлений /notifications

Список уведомлений с иконкой типа: ревью готово / дедлайн приближается / блокер у сотрудника / сотрудник готов к аутстаффу
Каждое уведомление: заголовок, текст, время, статус (прочитано / не прочитано)
Непрочитанные выделены визуально
Кнопка "Отметить все прочитанными"
Фильтры по типу уведомления
Счётчик непрочитанных в хедере (глобальный, через Zustand)


3. Профиль для аутстаффа /export/[id]

Страница-резюме конкретного сотрудника для отправки клиенту
Блоки: имя, стек (теги), опыт, readiness score, последние достижения, замечания ментора (только публичные)
Кнопка "Скопировать как текст" — копирует всё в буфер в читаемом формате
Кнопка "Печать" — window.print(), страница адаптирована под печать через CSS @media print
Список всех готовых к аутстаффу /export — таблица с фильтрацией по стеку и грейду


4. Настройки платформы /settings

Вкладки: Критерии оценки / Роли / Уведомления
Критерии оценки: список критериев для ревью (название, вес в %), можно добавить новый, удалить, изменить порядок
Роли: таблица пользователей с ролями (admin / mentor / viewer), смена роли через дропдаун
Уведомления: тогглы — какие события присылать уведомления (дедлайн за 3 дня, новое ревью и т.д.)
Валидация форм через Zod
